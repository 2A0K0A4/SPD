[SECTION 1: Problem Statement]
┌─────────────────────────────────────────────────────────────────────────┐
│ MOTIVATION                                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  Current speech processing systems face:                               │
│                                                                         │
│  • Dataset Heterogeneity: Different formats (MP3, FLAC, WAV) across   │
│    multiple sources (Common Voice, LibriSpeech, SAA, VoxPopuli)       │
│                                                                         │
│  • Metadata Inconsistency: Accents lack standardization, 50+ variants │
│    need categorization into coherent groups                           │
│                                                                         │
│  • Data Quality: No systematic validation of file integrity, audio    │
│    properties, or speaker distribution                                │
│                                                                         │
│  • Speaker Leakage: Naive random splitting includes same speakers in  │
│    train/test sets, invalidating model evaluation                     │
│                                                                         │
│  • Limited Accent Coverage: Existing datasets focus on English;       │
│    multilingual accents underrepresented                              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

[SECTION 3: Key Technical Features (Right Column, Top)]
┌─────────────────────────────────────────────────────────────────────────┐
│ KEY INNOVATIONS                                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  ✓ Multi-Format Support                                               │
│    Unified processing for 4 dataset types:                            │
│    - Common Voice (MP3 + TSV manifests)                               │
│    - LibriSpeech (FLAC + text transcripts)                            │
│    - Speech Accent Archive (WAV files)                                │
│    - VoxPopuli (MP3 + language structure)                             │
│                                                                         │
│  ✓ Speaker-Independent Splitting                                      │
│    Uses GroupShuffleSplit with speaker_id grouping                    │
│    → Prevents same speaker in train/test (data leakage)               │
│    → Ensures valid model evaluation metrics                           │
│                                                                         │
│  ✓ Comprehensive QA Framework                                         │
│    10-point validation checklist:                                     │
│    - File existence & integrity                                       │
│    - Duration matching & minimum thresholds                           │
│    - Transcript completeness                                          │
│    - Audio quality metrics                                            │
│    → 100% pass rate on pilot data                                     │
│                                                                         │
│  ✓ Accent Categorization                                              │
│    Intelligent mapping with linguistic basis:                         │
│    - Arabic: 6 variants (Egyptian, Levantine, Gulf, ...)              │
│    - South Asian: 5 variants (Indian, Pakistani, ...)                 │
│    - East Asian: 6 variants (Mandarin, Japanese, ...)                 │
│    - European: 8 variants (French, German, Spanish, ...)              │
│    - North American: 2 variants (American, Canadian)                  │
│                                                                         │
